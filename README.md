Intro_to_DB: ALX Backend Development - Introduction to Databases

This repository contains SQL and Python scripts for the ALX backend development curriculum focusing on relational database management using MySQL. The tasks involve schema design, database creation, and table definition.

Prerequisites

To run the Python and SQL scripts, you will need:

A running instance of a MySQL Server.

The mysql.connector Python library (or similar).

Usage Note

For MySQLServer.py, ensure you update the db_config dictionary with your local MySQL credentials (user and password) before execution.
